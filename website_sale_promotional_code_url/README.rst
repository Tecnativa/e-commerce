=================================
Website Sale Promotional Code URL
=================================

This module extends the functionality of website_sale module to show prices
based on a pricelist by promotional code in url

Usage
=====

To use this module, you need to:

#. Go to a pricelist and set promotional code and you con access to shop
writing 'http://base_url/shop/code/promotional_code'
